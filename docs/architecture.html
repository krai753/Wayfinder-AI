Wayfinder AI — System Architecture

Voice-first accessible flight booking for blind & visually impaired users

🎙️ Voice-First Pipeline

  • • User speaks → Web Speech API captures audio
  • • Whisper fallback for noisy environments
  • • LLM parses intent + fields + confidence score
  • • OpenAI TTS reads back results to user
  • • Continuous loop: listen → think → speak

🛡️ Safety & Confidence Gate

  • • Layer 1: Single-utterance confidence <85% → immediate CS
  • • Layer 2: 3 consecutive low-confidence retries → escalate
  • • Emergency override: "emergency", "stranded" keywords
  • • No booking is ever made on sub-85% confidence data
  • • Full context chain passed to human agent on escalation

🤝 Human-in-the-Loop

  • • CS Agent Dashboard with chat + call + book tabs
  • • City-name to IATA auto-resolution (London→LHR)
  • • Async messaging: agent sends text → TTS to user
  • • Real-time voice call between agent and user
  • • Ticket persists via SQLite across sessions
