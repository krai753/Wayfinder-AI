# Wayfinder AI - React Frontend

🎤 Voice-first, accessible flight booking interface for visually impaired travelers.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Features

✅ Voice-first interface  
✅ Accessible design (WCAG AA)  
✅ Beautiful glassmorphism UI  
✅ Smooth animations  
✅ Mobile optimized  
✅ Dark theme

## Tech Stack

- React 18 + TypeScript
- Vite (fast bundler)
- Tailwind CSS
- Motion (animations)
- Lucide React (icons)
